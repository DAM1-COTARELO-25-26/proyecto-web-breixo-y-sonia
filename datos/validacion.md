# Validación del archivo tablaDestinos.xml

## 1. Herramientas utilizadas

### Validación DTD
- Heramienta: XML Copy Editor
- Versión: wxWidgets 3.1.7

### Validación XSD
- Herramienta: XML Copy Editor
- Versión: wxWidgets 3.1.7

## 2. Proceso de validación

### Validación contra DTD
Comprobar si bien formado (tick azul)
después pulsar en Validar (tick verde)
